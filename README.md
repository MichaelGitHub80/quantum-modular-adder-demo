# VBE‑Style Modular Adder — Quantum Circuit Implementation
This repository contains a full implementation and explanation of the ***Vedral–Barenco–Ekert (VBE)*** style ***modular adder***, including:

- A working quantum circuit implementation
- A step‑by‑step walkthrough of the algorithm
- A complete Dirac‑notation description of every transformation
- Both scenarios:
  - ***Case 1***: $a + b < N$
  - ***Case 2***: $a + b \geq N$
- A concrete numerical example showing that the quantum circuit matches the classical modular result

The goal is to provide a clear, educational reference for how modular addition works at the quantum‑gate level.
